# Larva Core

Larva Core is the foundational system behind **Larva**, an offline-first personal AI designed as a workstation for thinking, planning, and execution.

Larva is not built as a traditional chatbot.  
It is designed as a structured intelligence system where intent, memory, and workflow come before language generation.

---

## Philosophy

Larva is built on a few core principles:

- **Offline-first by default**  
- **User owns their data and structure**
- **Structure before automation**
- **Intent-driven intelligence**
- **Clarity over convenience**

The goal is not to replace thinking, but to support it.

---

## What Larva Is

- A personal AI system
- A project and decision workspace
- A long-term memory and intent engine
- A modular, extensible architecture

---

## What Larva Is Not

- Not a chatbot clone
- Not a messaging or social app
- Not cloud-dependent by default
- Not focused on casual conversation
- Not designed for entertainment
